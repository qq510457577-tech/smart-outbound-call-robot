# 智能外呼机器人

AI 驱动的自动化智能外呼系统。

## 功能

- 📞 **自动外呼** — 批量号码自动拨打
- 🤖 **AI 语音对话** — 语义理解、多轮对话
- 🎯 **意图识别** — 自动分类客户意向
- 📊 **数据看板** — 外呼结果实时统计
- 🔄 **CRM 对接** — 通话记录同步
